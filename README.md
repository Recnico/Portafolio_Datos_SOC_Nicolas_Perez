## 📂 Proyectos Destacados (Enfoque Novared)

### 1. Análisis Forense y Reporte de Incidentes (SOC & Reporting)
*Este proyecto demuestra tu capacidad para elaborar informes y analizar tráfico, requisitos clave de la oferta.*
* **Descripción:** Simulación de ataques de red y análisis forense de paquetes para identificar patrones de intrusión.
* **Herramientas:** Wireshark, Hping3, Kali Linux.
* **Resultado:** Generación de un **Informe de Hallazgos y Mitigación**, detallando la vulnerabilidad encontrada y la estrategia de defensa.
* 🔗 [Ver Documentación del Proyecto](TU_LINK_A_GOOGLE_DRIVE_O_GITHUB)

### 2. Automatización de Procesamiento de Logs con Python
*Cumple con el requisito de "programación de scripts para procesamiento de datos".*
* **Descripción:** Desarrollo de scripts en Python para la limpieza y transformación de grandes volúmenes de datos. Aunque el ejemplo original es de E-commerce, se presenta como una metodología aplicada a **Logs de Seguridad**.
* **Herramientas:** Python (Pandas), Jupyter Notebooks.
* **Habilidad Clave:** Limpieza de datos (Data Cleaning), manejo de outliers y estructuración de datasets para análisis de tendencias.
* 🔗 [Ver Notebook en GitHub](https://github.com/Recnico/EDA-ComercioYA/tree/main)

### 3. Dashboard de Métricas y Tendencias (Power BI)
*Alineado con "análisis de datos y tendencias".*
* **Descripción:** Creación de dashboards interactivos para la visualización de KPIs. (Aquí puedes mencionar que esta misma metodología la aplicas para visualizar alertas de seguridad en un SOC).
* **Herramientas:** Power BI, Power Query, DAX.
* **Resultado:** Visualización clara de tendencias que permiten la toma de decisiones basada en datos.

### 4. Implementación de Seguridad Perimetral y SIEM
*Demuestra conocimiento en plataformas de seguridad (Firewalls, SIEM).*
* **Descripción:** Configuración de topologías seguras incluyendo Firewall Cisco ASA (DMZ, VPN) y monitoreo mediante Wazuh SIEM.
* **Herramientas:** Cisco ASA, Wazuh, Syslog.
* **Habilidad Clave:** Correlación de eventos y monitoreo proactivo.
